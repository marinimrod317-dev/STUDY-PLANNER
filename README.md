# Study Planner

## Project Overview
This study planner is a web application that helps students to organize their tasks and track their progress.
Users can view tasks, mark the completed tasks, delete tasks, add new tasks and check progress.
This app manages time efficiently.

## Technologies Used
HTML
CSS
Javascript

## Features
Display all tasks
Mark completed tasks
Delete tasks
Check progress

## How to run
Add task via the Add Task page(task.html) and saved in localstorage.
The Dashboard page(index.html) displays all tasks.
Users can mark complete or delete tasks.
The progress section updates automatically.

## Known Issues
Task are only saved in localstorage and are not shared across devices
